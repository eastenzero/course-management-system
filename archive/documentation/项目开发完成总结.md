基于Django + React的课程管理系统已完成主要功能开发，系统包含完整的后端API、前端界面、数据分析和智能排课功能。

## 技术架构

### 后端技术栈
- **框架**: Django 4.2 + Django REST Framework
- **数据库**: SQLite（开发环境）
- **认证**: JWT Token认证
- **API文档**: DRF自动生成的API文档

### 前端技术栈  
- **框架**: React 18 + TypeScript
- **UI库**: Ant Design 5.x
- **图表库**: ECharts 5.x
- **构建工具**: Vite
- **状态管理**: React Hooks

### 核心模块
1. **用户管理** (`apps.users`)
2. **课程管理** (`apps.courses`) 
3. **教室管理** (`apps.classrooms`)
4. **排课管理** (`apps.schedules`)
5. **数据分析** (`apps.analytics`)

## API接口总览

### 认证接口
- `POST /api/v1/auth/login/` - 用户登录
- `POST /api/v1/auth/refresh/` - Token刷新

### 核心业务接口
- `GET /api/v1/users/` - 用户列表
- `GET /api/v1/courses/` - 课程列表  
- `GET /api/v1/classrooms/` - 教室列表
- `GET /api/v1/schedules/` - 排课列表

### 分析接口
- `GET /api/v1/analytics/overview/` - 概览数据
- `GET /api/v1/analytics/dashboard/stats/` - 仪表板统计
- `GET /api/v1/analytics/courses/` - 课程分析
- `GET /api/v1/analytics/users/` - 用户分析

### 智能排课接口
- `POST /api/v1/schedules/auto-schedule/` - 自动排课
- `POST /api/v1/schedules/optimize/` - 排课优化
