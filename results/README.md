# 🎓 智能大学课程表调度系统 - 结果文件说明

## 📊 项目概览
本系统成功实现了支持**数千人规模**的大学课程表调度，涵盖大学四年八个学期的完整教学管理流程。

## 📁 文件结构

```
results/
├── 📊 Excel报表文件
│   ├── comprehensive_timetable.xlsx      # 📈 主要排课结果报表
│   └── university_timetable_system.xlsx  # 📋 基础数据报表
├── 🗄️ 数据库文件
│   └── university_data.db                # 💾 SQLite完整数据库
└── 📜 SQL脚本文件
    ├── university_data.sql               # 🏗️ 基础数据结构
    └── timetable_data.sql                # 📅 排课数据导出
```

## 📊 Excel报表详细说明

### 🎯 comprehensive_timetable.xlsx（主要结果）

**工作表列表：**

1. **📋 总体课程表**
   - 完整的排课结果，包含所有学期、课程、教师、班级、教室安排
   - 字段：排课ID、学期、课程名称、教师、班级、教室、教学楼、星期、时间、周次、日期

2. **👨‍🏫 教师工作量统计**
   - 每位教师在各学期的教学任务统计
   - 字段：教师姓名、部门、职称、学期、授课门数、授课班数、总教学时数、总课时数

3. **🏢 教室使用统计**
   - 各教室的使用情况和利用率分析
   - 字段：教学楼、教室、容量、类型、使用次数、课程数、学期数

4. **📚 课程分布统计**
   - 按课程类别统计的分布情况
   - 字段：课程类别、类别名称、课程数、任务数、学生总数、总学时

5. **📈 数据汇总**
   - 核心统计指标汇总
   - 包含：教学任务总数、排课记录总数、参与教师数等关键指标

## 🗄️ 数据库文件说明

### 💾 university_data.db
SQLite数据库文件，包含完整的大学教学管理系统数据：

**主要数据表：**
- `colleges` - 学院表
- `majors` - 专业表  
- `classes` - 班级表
- `students` - 学生表 (2,880条记录)
- `teachers` - 教师表 (250条记录)
- `courses` - 课程表 (1,040条记录)
- `rooms` - 教室表 (99条记录)
- `timetable` - 课程表 (8,736条记录)
- `teaching_tasks` - 教学任务表 (1,950条记录)

**查看方法：**
```bash
# 进入SQLite命令行
sqlite3 university_data.db

# 查看所有表
.tables

# 查看数据统计
SELECT '学生总数' as 指标, COUNT(*) as 数量 FROM students
UNION ALL
SELECT '教师总数', COUNT(*) FROM teachers
UNION ALL
SELECT '课程总数', COUNT(*) FROM courses
UNION ALL
SELECT '排课记录', COUNT(*) FROM timetable;
```
