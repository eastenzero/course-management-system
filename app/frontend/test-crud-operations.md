# 增删改查功能测试指南

## 已实现的功能模块

### ✅ 1. 课程管理 (完整CRUD)

#### 查看 (Read)
- **课程列表**: http://localhost:3000/courses/list
  - 支持分页 (20条/页)
  - 支持搜索 (课程名称、代码)
  - 支持筛选 (院系、课程类型)
  - 显示600门真实课程数据

#### 创建 (Create)
- **创建课程**: http://localhost:3000/courses/create
  - 表单字段：课程代码、名称、学分、院系、课程类型、最大学生数、总学时、学期、学年、描述
  - 实时验证和错误提示
  - 成功后跳转到课程列表

#### 编辑 (Update)
- **编辑课程**: http://localhost:3000/courses/{id}/edit
  - 自动加载现有数据
  - 支持所有字段修改
  - 保存后跳转到课程详情

#### 详情 (Read)
- **课程详情**: http://localhost:3000/courses/{id}
  - 显示完整课程信息
  - 包含创建时间、更新时间
  - 显示当前选课人数

#### 删除 (Delete)
- **删除课程**: 在课程列表页面点击删除按钮
  - 确认对话框
  - 删除后自动刷新列表
