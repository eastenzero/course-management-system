## 组件规范总览

- 目标：在不改业务的前提下，为现有组件提供玻璃质感、低饱和主题、明确态反馈与无障碍支持
- 适用范围：按钮、输入类、选择类、状态与反馈类、数据展示类、导航与容器类

---

## 通用规则

- 四态：默认 / 悬浮 / 按下 / 禁用（Light/Dark 两套）
- 结构：图标 + 文本的对齐与间距一致；图标大小统一（例如 16/20px）
- 玻璃表面：按层级使用 blur-sm/md/lg；alpha 随明暗变化
- 焦点可见：键盘焦点使用 2px 高对比描边或外发光

---

## 按钮 Button

- 形态：主按钮 Primary、次按钮 Secondary、幽灵 Ghost、危险 Danger
- 圆角：10–12；内边距：水平 12–16，垂直 8–10
- 玻璃：
  - Primary：玻璃表面 + 主题色渐变微弱（可选）
  - Ghost：透明玻璃 + 细描边
- 四态示意（以 Light 为例）：
  - Default：alpha 0.60，细描边 α0.25
  - Hover：提亮 + 阴影增强
  - Active：轻压缩 0.98 + 阴影收敛
  - Disabled：降低饱和与对比，禁用指针

---

## 输入与表单（Input / Textarea / Select / Switch）

- 输入框：
  - 玻璃表面 alpha 0.60；占位 Text-3；聚焦时描边与高光提升
  - 错误态：柔和红描边 + 清晰帮助文案
- 选择器/下拉：
  - 弹出层模糊提升一级；选中行背景轻微提亮，支持键盘导航
- Switch：
  - 胶囊玻璃轨道；打开时主题色柔光；禁用时降低对比

---

## 标签与徽章（Tag / Badge）

- 低饱和填充或细描边；圆角胶囊；批量操作支持批选态
- 徽章数字使用等宽或居中对齐，避免跳动

---

## 反馈（Toast / Tooltip / Notification / Empty）

- Toast/通知：
  - 玻璃表面 + 高光描边；进入 200–250ms，停留 2–4s
  - 成功/警告/错误/信息 4 种语义色
- Tooltip：
  - 轻模糊 + 细描边；三角箭头不投硬阴影
- 空状态 Empty：
  - 插画 + 引导文案 + 一键操作（如“创建课程”）

---

## 数据与可视化（Card / Table / Progress / Chart）

- 卡片 Card：
  - 圆角 16；玻璃 alpha 0.60；阴影 md，Hover 时提升亮度与阴影
- 表格 Table：
  - 弱化网格线（极细分隔）；Hover 行背景提亮；固定表头半透明
  - 大数据集滚动性能优化，禁用过度阴影
- 进度 Progress：
  - 进度条轻玻璃轨道；完成时给予正反馈动画
- 图表 Chart：
  - 低饱和配色；关键数据点加强调色；工具提示玻璃化

---

## 导航与容器（Navbar / Sidebar / Tabs / Modal / Drawer）

- 顶栏 Navbar：
  - 玻璃横条 + 模糊；滚动吸顶时阴影增强
- 侧栏 Sidebar：
  - 玻璃分组；当前项柔和高亮背景 + 高光描边
- Tabs：
  - 胶囊切换；活动项下划线或高光描边
- Modal/Drawer：
  - 背景模糊遮罩；面板 20–24 圆角；进入缩放 + 模糊淡入

---

## 微交互与动效参数

- 时长：进入 200–250ms，退出 150–200ms
- 缓动：cubic-bezier(0.22, 1, 0.36, 1)
- 拉伸/压缩：Active 压缩 0.96–0.98，完成回弹
- Skeleton：低频渐变，占位颜色随主题中性层

---

## 无障碍与可用性

- 对比度 AA：正文 4.5:1，标题 3:1
- 键盘导航：Tab/Shift+Tab 完整覆盖；焦点环明显
- 颜色非唯一：状态同时用图标/文案表达
- 减少动效：系统“减少动效”开启时关闭非必要动画

---

## 组件验收清单（抽样）

- 外观：圆角/阴影/玻璃 alpha/描边符合规范
- 态：四态可见且差异明确
- 可读性：文本与图标对比度达标
- 性能：滚动、打开弹层无明显卡顿
- 无障碍：焦点可见、ARIA/语义正确
